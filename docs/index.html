Claude Sessions Documentation

Welcome to Claude Sessions - the open source session archiver for Claude Code. Never lose context again.

What is Claude Sessions?

Claude Sessions automatically archives your Claude Code conversations, making them searchable and resumable. It uses Claude Code's hook system to capture sessions when you exit, extracts key information, and stores everything in a local SQLite database.

Open Core Model

Claude Sessions follows an open core model: the core functionality is free and open source forever, while advanced features for teams and power users are available in Claude Sessions Pro.

Free & Open Source

Claude Sessions Core

Everything you need for personal session management. Auto-archive, search, checkpoints, and web dashboard.

Pro

Claude Sessions Pro

Access your sessions from anywhere. Cloud backup, multi-device sync, and team collaboration.
