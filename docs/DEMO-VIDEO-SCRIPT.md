## Arcade.software Tips

1. **Use Speed Controls**:
   - Real-time for important moments
   - 2x-4x for installation/loading

2. **Add Hotspots**:
   - Highlight the install command
   - Highlight search results
   - Highlight key UI elements

3. **Use Annotations**:
   - Add arrows pointing to important output
   - Use callout boxes for explanations

4. **Terminal Setup**:
   - Use a clean terminal theme (dark background)
   - Increase font size for readability
   - Hide irrelevant shell prompts/paths

5. **Browser Setup** (for web dashboard):
   - Use a clean browser profile
   - Hide bookmarks bar
   - Maximize the window

## Recording Checklist
