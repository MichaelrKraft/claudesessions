Save a snapshot of this current Claude Code session:

1. Run the save-now script to archive the current session state:
   ```bash
   ~/.claudesessions/bin/save-now.sh $ARGUMENTS
   ```

2. The argument (if provided) becomes a tag/name for this session

3. Confirm to the user their session was saved

4. Remind them they can continue working - this is just a checkpoint

This creates a searchable archive they can browse later with /archives.
