Browse archived Claude Code sessions:

1. Run the sessions CLI to show archives:
   ```bash
   ~/.claudesessions/bin/sessions $ARGUMENTS
   ```

2. Available subcommands:
   - list: Show all archived sessions
   - search <query>: Find sessions containing text
   - view <name>: View session details
   - stats: Show archive statistics

3. Help the user search, view, or export any session they're interested in
